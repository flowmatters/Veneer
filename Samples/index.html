Instructions for tesing out the Web server / reporting components

  1. Unzip the Web Server plugin into your Source\Plugins folder (this version is built against 3.2.6.33 and will only work with that version
  2. Run a Windows Command Prompt as administrator. (For example, in Windows 7, press the Windows key then type cmd into the search box, then right click on the cmd icon and select run as administrator
  3. In the command prompt, run the command netsh http add urlacl url=http://+:9876/ user=joel (substituting in your windows user name
  4. Start source and load the flowmatters.source.webserverpanel.dll plugin (before loading any project).
  5. Load a catchments project.
  6. Load the Web Server from Tools|Plugins|FlowMatters.Source.WebServerPanel|Web Server Monitoring. You should see a window appear with the message Started Source RESTful Service
  7. Start Firefox (or possibly IE10, but NOT Chrome!) and load this file

Examples

  • Simple time series graph (hard coded to work for an AWBM example project) (D3.js)
  • List and view scenario results (D3.js)
  • Select and view time series with interactive google chart
  • Simple pie chart showing contribution of sediment from each catchment (expects a constituent named TSS)
